The tools ecosystem is a series of lightweight software to create audio-visual works.

Each tool is single-file implementation written in portable ANSI C using SDL2 with a lightweight front-end designed around the chr_format. You can learn more about the philosophy behind these various projects on the Hundred Rabbits blog.

The goal of this ecosystem would be to eventually migrate the tools to an original virtual machine, and allow people to download roms for its emulator. The work toward this goal is recorded under the Uxn project.
