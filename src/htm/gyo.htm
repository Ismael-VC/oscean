Gyo is a single-cycle Forth virtual machine.

This virtual machine was created as a result of experimenting with stack-effect checking and so was designed around the idea that each opcode is encoded with its own stack effect. The layout of the opcodes is inspired by the NOVIX NC4016 and James Bowman's Forth J1 CPU.
