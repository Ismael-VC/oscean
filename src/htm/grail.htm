Grail is a hand-writing recognition program inspired by the program from RAND by the same name.

Grail is a graphical input system to input characters with a pen or d-pad controller in which every character is reduced to a sequence of direction changes in the path that makes up a letter or number. It supports the basic ASCII numbers, letters and symbols.

Special Operations
0x06begin selection
0x11up
0x12down
0x13left
0x14right
0x16end selection
