Collected doodles made in a 4000 bytes program.

Oekaki is a somewhat incapable 2-bit drawing program for Varvara inspired from Noodle, with very even fewer features. The default chr image size is 40x28, or 512x320. It can open tga images. It was originally created to do close-ups for scenes of the Neauismetica.

Controls

  • ctrl+n New
  • ctrl+s Save
  • ctrl+o Open
  • tab Toggle menu
  • [] Brush size
  • 1234 Select color
  • qwer Select shape
  • asdf Select tool
  • h Flip horizontal
