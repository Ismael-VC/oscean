Yes, a potato.

Potato is an operating system for the Varvara computer, designed to fill the gaps where a host device might not have an underlying file system, like handheld consoles.

The majority of the controls are designed around the d-pad. The pattern used here is the B button to handle all window operations, and reserve the A button for the applications themselves. Files can also be opened with mouse2.

  • B+Up expand a window
  • B+Down tab a window
  • B+Right open a window
  • B+Left close a window

Applications

From Potato, you can jump into a rom, and return to the OS with f4.

  • Tile: A tile editor, used to modify the system's icn/chr assets.
  • Color: A color slider, used to modify the theme.
  • Picture: A picture viewer for 1-bit and 1-bit graphics.
  • System: A preview of the system's storage.
  • Text(TBA): A text editor.
  • Hexed(TBA): A hexadecimal editor.
  • Console(TBA): A cli terminal.
  • Play(TBA): A pcm player.
