Yes, a potato.

Potato is an operating system for the Varvara computer. It is designed to fill the gaps where a device might not have an underlying operating system, for example handheld consoles.

It features most window manager capabilities, although somewhat limited in scope. The majority of the controls are designed around the d-pad. The pattern used here is the B button to handle all window operations, and leave the A button to the applications. Alternatively, you can open a file with mouse2.

  • B+Up expand a window
  • B+Down tab a window
  • B+Right open a window
  • B+Left close a window

From Potato, you can jump into a rom, and return to the OS. It supports all the Uxn generics, such as the .theme, and .snarf files.
