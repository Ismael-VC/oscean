UFX is a 1-bit proportional font format for the Uxn system.

The UF* file contains a page of 256 bytes corresponding to the width of each of the 255 characters in the font (in pixels), followed by the pixel data in the .chr format for each character.

ExtensionSize(px)Filesize
.uf18x8(1 tile)0x0900
.uf216x16(4 tiles)0x2100
.uf324x24(9 tiles)0x4900

You will find this filetype in the Uxn ecosystem, namely in Left.
