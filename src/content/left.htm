Left is a text editor.

Left is graphical plain-text editor inspired by Plan9's Acme and THINK Pascal. It supports Uxntal syntax highlight, proportional text, inline images, a snarf buffer and chorded controls. The alt key is used as a LEAP key to navigate the body of the text.

Inline Graphics

Left has a tiny inline graphics format used to draw 1-bit icons and pictures, in which each byte is used to represent 4 vertical pixels. The X bit advances the rendering by 1 pixel horizontally, the Y bit shifts the pixels drawing by 4 pixels vertically for a total of 16 vertical pixels. Glyphs can be generated in Noodle.
