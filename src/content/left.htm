Left is a text editor.

Left is graphical plain-text editor inspired by Plan9's Acme and THINK Pascal. The Uxn version features Uxntal syntax highlight, proportional text, a snarf buffer and chorded controls. The alt key is used as a LEAP key.

A web version was initially created to help Rekka with the writing of Wiktopher, and featured auto-completion, synonyms suggestions and writing statistics.
