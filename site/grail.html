    Grail is a graphical input language.

    Grail is a graphical input system to input characters with a pen or d-pad controller in which every character is reduced to a sequence of direction changes in the path that makes up a letter or number. It supports the basic ASCII numbers, letters and symbols. This hand-writing recognition program was directly inspired by the program from RAND by the same name.

    Usage

    The drawing surface is split into three regions, the upper one capitalizes the letters, the middle one is lowercase letters and the bottom one is punctuation. See the full collection of supported glyphs.

    Extras

    Grail includes arrow controls, sent as bytes. To use this mode, you need a pen with buttons for right-click. This scheme is supported in Left.

    Special Operations
    0x06begin selection
    0x11up
    0x12down
    0x13left
    0x14right
    0x16end selection
