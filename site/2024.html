    2024-01-13 Maintenance

    The forge that we use at Hundred Rabbits has been taken down by DDoS attacks and is struggling to come back online, the event reminded us that we ought to also have mirrors and release versions of these source files elsewhere.

    Until we regain access and push the changes of the last few days, keeping with the spirit of improving the resilience of the tools we use I've taken a moment to write a kind of pocket version of the console emulator and self-hosted assembler as to see how many lines are needed to start from the seed assembler and replicate it. A copy of the pocket emulator, the source for the assembler and its hexadecimal representation have been added to the wiki.
