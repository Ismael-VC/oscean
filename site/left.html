    Left is a text editor.

    Left is graphical plain-text editor inspired by Plan9's Acme and THINK Pascal. It supports Uxntal syntax highlight, proportional text, inline images and the snarf buffer. The alt key is used as a LEAP key to navigate the body of the text.

    Manual

    Left doesn't do much, but packs many useful features in a tiny 13kb rom.

    • ctrl+up next label
    • ctrl+down prev label
    • ctrl+left end of line
    • ctrl+right start of line
    • alt+left next word
    • alt+right prev word

    Finding

    Finding a word is done by holding down the alt key, and typing characters, and releasing alt to jump. Alternatively, you can select a word with mouse2, and jump to the next instance with mouse3, you can change the direction of the jump with alt+tab.

    • mouse1 select character
    • mouse2 select word
    • mouse3 find

    Directory

    A listing of the files in the active directory can be seen by the sequence ctrl+r del ., a file from that listing can be opened by the sequence mouse2 ctrl+g.

    • ctrl+n new
    • ctrl+s save
    • ctrl+o open
    • ctrl+g open selection
    • ctrl+r rename
    • ctrl+c copy
    • ctrl+v paste
    • ctrl+x cut
    • ctrl+h toggle syntax
    • ctrl+f toggle font
