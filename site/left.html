Left is a text editor.

Left is graphical plain-text editor inspired by Plan9's Acme and THINK Pascal. It supports Uxntal syntax highlight, proportional text, a snarf buffer and chorded controls. The alt key is used as a LEAP key to navigate the body of the text.

A web version was initially created to help Rekka with the writing of Wiktopher, and featured auto-completion, synonyms suggestions and writing statistics.

Inline Graphics

Left has a tiny inline graphics format used to draw custom icons and small graphics, where each byte is used to represent 4 vertical pixels.
