    A certain enchanted forest is inhabited by talking birds.

    Combinatory logic is a set of primitives from which numbers, functions and computability can emerge. Combinatory Logic contains, in its most basic form, special operators which we will call birds, along with parentheses and symbols which themselves also represent other birds.

    We always start from the left, applying the rule for the left-most combinator first. We can omit some parentheses and f(x) may be written simply as fx.

    ▲ The Abemet Forest

    Within the Abemet forest, we find the A, B, M, and T birds. Each bird listens to a message, and replies with another.

    The first bird that we observe is the Mockingbird, which repeats the first word.

    M▲◆● -> ▲▲◆● W(CKC)xyz

    The second bird that we observe is the Albatross, which omits the first word.

    A▲◆● -> ◆● K(CKC)xyz

    The third bird that we observe is the Thrush, which swaps the first and second words.

    T▲◆● -> ◆▲● C(CKC)xyz

    We can't quite make out the fourth bird, but we observe that it is the Bluebird, which is also found in the next forest, so let's continue.

    ◆ The Becekew Forest

    Within the Becekew forest, we find the B, C, K, and W birds.

    The first bird that we observe is the Warbler, which repeats the second word. This bird's song is similar to the Mockingbird, which repeats the first word.

    W▲◆● -> ▲◆◆● Wxyz

    The second bird that we hear is the Kestrel, which always omits the second word. This bird's song is similar to the Albatross, which omits the first word. Here is the Kestrel, which always omits the second word, followed by the Warbler, which repeats the second word.

    KW▲◆● -> W◆● -> ◆●● KWxyz

    The third bird that we hear is the Cardinal, which swaps the second and third words. This bird's song is similar to the Thrush, which swaps the first and second words. Here is the Cardinal, followed by the Kestrel:

    CK▲◆● -> K◆▲● -> ◆● CKxyz

    The elusive Bluebird takes three arguments, and calls the first argument with the result of calling the second argument with the third. If you are not familiar with function composition, this means chaining functions in such a way that the return value of one is passed to the next, like x(y(z)).

    Bluebirdx (y z)Composition of x and y applied to z
    Cardinalx z ySwaps the second and third arguments
    Kestrelx zDiscards the second argument
    Warblerx y y zDuplicates the second argument

    One of the things BCKW has over SKI is that each combinator represents a concrete capability. In BCKW, each bird seems to skip over its first argument, and work on its second, the reason for this choice of birds is that while it is easy to form operations that operate on their first arguments, it is extremely difficult to go the other way and work from primitives using their first argument and form the BCKW primitives.

    ● The Seki Forest

    The simplest of these birds, appears to have no intelligence at all, and so it is called the Idiot bird since its response to hearing the name of any bird is that same bird. 

    λxyz -> x(y)(z)identity combinator

    The Kestrel's response to hearing a song is a bird which responds with its first parameter, no matter what it hears. The Idiot bird can also be defined as SSK, the identity function, since this derivation works for any expression X by SKK is also X.

    λxyz -> x(z)constant-making combinator

    The Starling does parenthesising, reordering, and duplication. Applied to three combinators, it applies each of the first two combinators to the last, then applies the first result of that to the second:

    λxyz -> x(z)(y(z))application combinator
    Starlingxyzxz (y z)Returns the first argument applied to the third, which is then applied to the result of the second argument applied to the third.
    Kestrelx yxDiscards the second argument
    IdiotxxReturns its argument

    Summary

    • I := λx.x
    • S := λx.λy.λz.x z (y z)
    • K := λx.λy.x
    • B := λx.λy.λz.x (y z)
    • C := λx.λy.λz.x z y
    • W := λx.λy.x y y
