    Bouc is a text reader.

    Sometimes I just want to read something quietly on the computer without having to contend with browser notifications or fight with Calibre's rendering artifacts when I'm reading a simple text.

    It uses the snarf buffer to remember the position in a file. It supports theme files for customization.

    That's it, it reads text files.
