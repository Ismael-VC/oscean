    December's Advent Of Ure.

    In lieu of doing the Advent of Code, I aim to do a little bit of software programming every day in December. Possibly implement a handful of fun X11 software to Uxntal, improve existing roms and document the daily progress.

    01 DEC Completed the design of the panel for Blinkenlights. This little Uxn emulator includes a knob to change the evaluation rate, a stepper and a button to start the program over.

    02 DEC Added a much needed 8x8 zoom grid to Noodle, which will make it a lot easier to do quick alterations to 1-bit tilesets .

    03 DEC Created a little calendar application which has been a wonderful exercise in learning about the routines needed in finding the day in the week of a given date, finding the number of days a month, and finding the number of days in a year.

    04 DEC Transformed the little calendar program, into a kind of virtual whiteboard onto which I can scribble notes overlaying the calendar. It records each month's drawings as icn files, making it possible to edit them with Noodle.

    05 DEC It's a busy day aboard Pino today, so I couldn't pick as large a task as with the previous days, so I've decided to make my pixelart life slightly more pleasant by adding a selection size label to remove some of the guess work in Nasu.

    06 DEC I decided to repurpose the emulator at the core of Blinkenlights, and turn it into something that would be more practical. Inspired from Uxn32's Beetbug, I've made a stand-alone debugger that works on all emulators and allow anyone to step through their program.

    07 DEC I realized that I might have built something invaluable yesterday with the native implementation of Beetbug, I've spent my morning adding controller input and breakpoints to this little rom which will now be part of my permanent collection of utilities.

    08 DEC I wanted to push this little debugger further yet, but to do so, I had to make Drifblim write a symbol file, and Beetbug read it. The running program's instructions are now addorned by little labels from the assembly file.

    09 DEC Unfortunately, I do not have a screenshot to show for today's software project but I did draw a big turnip, so. So you'll have to take my words for it, taking the symbols file implementation detail, with the help of @tbsp, I've added symbols output to uxnasm. Next step will be to add support symbols in the error reporting of uxncli and uxnemu, something for tomorrow.

    10 DEC The day went by very fast indeed, between Uxn ending up on the front-page of Hackernews, and answering the emails that ensued, compounded by the release of a most incredible implementation of a Gameboy emulator for Varavara, it was soon noon and I had to resume the rewiring of the boat. But, before the day was over, I managed to sneak in a few improvements to Dexe which I have been using daily.

    11 DEC I had some projects I wanted to do today, at least I think I did, but I found a nasty bug in uxnlin that really annoyed me, to the point that I dropped whatever it was I was doing and spent the whole morning fixing it.

    12 DEC I implemented metadata to Orca, and added 1-bit sprite-copying to Nasu, so I can pull assets without having to manually erase the extra color channel.

    13 DEC I've implemented a limited version of the calendar routine I wrote at the start of the month, into Potato!
