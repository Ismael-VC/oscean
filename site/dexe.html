    Dexe is a hex editor.

    Dexe is a graphical hex editor created to inspect and edit binary files, it displays bytes, ascii characters as well as sprites in the .icn format. It supports the theme and snarf files.

    It is currently used in the creation of Famicom and Uxn projects, it's an ideal tool to inspect or extract assets from assembled roms.
