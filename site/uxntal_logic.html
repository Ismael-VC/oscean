    Uxntal Logic

    Comparison operators will always push 01 for true, 00 for false. This is also true for operators in the short mode.

    #12 #34 EQU 00
#1234 NEQ 01
#1234 #7856 LTH2 01

    A non-null byte can be fed to the JCN opcode to create a for-loop:

    #10 #00
@loop
	( body, will be ran 16 times )
	INC GTHk ,loop JCN
POP2

    It can also be used to create something like a while loop:

    ;word
@while
	LDAk #18 DEO
	INC2 LDAk ,while JCN
POP2
@word "vermillion $1
