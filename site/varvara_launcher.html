    The varvara launcher is a way to navigate between programs in Varvara.

    The launcher is a rom used to jump between roms. A stock launcher.rom file should be provided with the emulator, and loaded automatically if present in the same folder.

    This rom should at allow to navigate between roms that are available in the current folder. If no rom is loaded into the emulator, the launcher should start. The basic launcher allows to assemble uxntal files.
